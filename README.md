# Customer-Segmentation

## Overview
This project segments customers based on transaction data using K-means clustering. The goal is to group customers with similar purchasing behaviors to assist the retailer in targeted marketing and sales strategies.

## Steps:
1. **Load the Data**: Import transaction data from the provided dataset.
2. **Data Preprocessing**: Clean and aggregate the data based on customer transactions.
3. **Feature Engineering**: Create features such as total sales, purchase frequency, and most bought items.
4. **Normalization**: Standardize data to ensure consistent scaling.
5. **K-Means Clustering**: Determine the optimal number of clusters using the Elbow Method and apply K-means for segmentation.
6. **Visualization**: Use pairplots to visualize customer clusters.

## Conclusion
The clustering model helps identify customer segments that can be targeted with customized marketing strategies.
